# Lab - Decoders and Multiplexers  
This folder contains the implementation of Lab 1-2 for the CS 223 Digital Design Laboratory course.
## Objective
The goal of this lab is to understand and implement fundamental digital building blocks such as multiplexers and decoders using SystemVerilog.
## Contents
This folder contains the SystemVerilog implementations prepared for Lab 2.  
The codes include multiplexer, decoder, logic function, and bit shifter designs required in the assignment.

- Function g: Implements the given truth table by using a multiplexer and minimum additional logic.
- Logic Function F: Implements the required Boolean function using one 8-to-1 multiplexer and one 2-to-4 decoder.
- Bit Shifter: Shifts a 4-bit input right by one position when the control signal is active.
- Testbenches: Used to simulate and verify the correctness of each design.
- Implementation of 2-to-4 and 3-to-8 Decoders and 3-to-1, 4-to-1 and 8-to-1 Multiplexers
